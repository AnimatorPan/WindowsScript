# 视频转GIF工具 v1.0

一个Windows桌面应用程序，用于将视频文件转换为GIF动画。

## 功能特性

- **视频文件选择**: 支持选择本地视频文件（MP4、AVI、MOV、MKV等常见格式），支持拖拽文件到界面
- **时间范围选择**: 可选择转换的时间范围，精确到毫秒
- **视频区域裁切**: 在视频预览画面上框选任意区域进行裁切，可视化裁切框支持拖拽调整位置、大小
- **输出参数配置**: 
  - 输出尺寸：宽度/高度设置（保持宽高比选项）
  - 帧率(FPS)：8-30fps可选
  - 颜色质量：256色/128色/64色/32色
- **GIF文件优化**: 支持4种优化级别（高质量/标准/压缩优先/极致压缩），文件大小预估
- **预设功能**: 保存常用配置为预设，快速加载
- **后台转换**: 转换过程不阻塞UI，支持进度显示和取消

## 安装说明

### 方法一：使用可执行文件（推荐）

1. 下载发布版本的可执行文件
2. 双击运行即可

### 方法二：从源码运行

1. 安装Python 3.8+
2. 安装依赖:
   ```bash
   pip install -r requirements.txt
   ```
3. 下载FFmpeg:
   ```bash
   python download_ffmpeg.py
   ```
4. 运行程序:
   ```bash
   python main.py
   ```

### 方法三：打包为可执行文件

1. 完成方法二的所有步骤
2. 运行打包脚本:
   ```bash
   python build_exe.py
   ```
3. 打包后的可执行文件位于 `dist/视频转GIF工具.exe`

## 使用说明

1. **选择视频**: 点击"浏览..."按钮选择视频文件，或直接将视频文件拖拽到窗口
2. **设置时间范围**: 调整"开始"和"结束"时间，选择要转换的视频片段
3. **裁切区域**: 在视频预览画面上拖拽白色框体选择要保留的区域，或使用右侧的数值输入框精确调整
4. **设置输出参数**: 
   - 选择输出尺寸（高度设为0则自动保持宽高比）
   - 选择帧率（数值越高动画越流畅，文件越大）
   - 选择颜色质量和优化级别
5. **选择输出路径**: 点击"浏览..."按钮选择GIF保存位置
6. **开始转换**: 点击"开始转换"按钮，等待转换完成

## 裁切框操作

- **拖拽框体**: 移动裁切位置
- **拖拽手柄**: 调整裁切大小（8个白色小方块）
- **双击框体**: 重置为全视频区域
- **滚轮**: 缩放裁切框（保持中心点）

## 预设说明

预设是保存的常用配置组合，方便快速切换：

- **高质量**: 20fps, 256色, 适合色彩丰富的视频
- **标准**: 15fps, 128色, 平衡质量和文件大小
- **压缩优先**: 10fps, 64色, 较小文件，适合分享
- **极致压缩**: 8fps, 32色, 最小文件，适合简单动画

可以保存自定义预设，但默认预设不能修改或删除。

## 技术栈

- **GUI框架**: PyQt6
- **视频处理**: FFmpeg
- **GIF生成**: FFmpeg (使用crop滤镜 + 调色板优化)
- **打包工具**: PyInstaller

## 注意事项

1. 首次使用需要下载FFmpeg，运行 `download_ffmpeg.py` 脚本
2. 转换大文件或高分辨率视频可能需要较长时间
3. 裁切区域的最小尺寸为10x10像素
4. 输出高度设为0时，会自动根据宽度计算保持宽高比
